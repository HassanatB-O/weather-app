<h2>Weather App (Tkinter + OpenWeather API)</h2>

<p>This is a simple desktop Weather Application built with Python, Tkinter, and the OpenWeather API.</p>
<p>The app allows users to enter a city name and fetch real-time weather information such as temperature and weather description.</p>

<h2>Technologies Used</h2>

<ul>Python</ul>
<ul>Tkinter – for building the desktop GUI</ul>
<ul>Requests – for making HTTP requests to the weather API</ul>
<ul>OpenWeather API – for weather data</l>
